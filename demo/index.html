Usage

Include the scripts and styles in your HTML head:

<link rel="stylesheet" href="tip.css" type="text/css" />

<script src="tip.js"></script>

Then simply add a title and class "tip-js" to any element you want to add Tip to:

<a href="#link" class="tip-js" title="This is the tooltip content">
  A Link
</a>

Then simply hook up TipTop:

<script>
window.onload = function(){
    var obj = new tipJs();
};
</script>

Advanced

Tip.js has two settings:

  • offsetY - Vertical offset in px (defaults to 10)
  • offsetX - Horizontal offset in px (defaults to 10)

Browser Compat

Tip.js works in all modern browsers (Chrome, Firefox, Safari, Opera) and probably in IE8+.
